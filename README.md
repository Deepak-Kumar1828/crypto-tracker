# 💰💰 crypto-tracker
A simple, clean crypto watchlistweb app build with Flask. Track live cryptocurrency prices, manage a personalized watchlist, and visualize prices with an interactive chart - all in a lightweight Python + Flask stack.

## Features 
-Add any cryptocurrency to your watchlist (using CoinGecko coin IDs)
-View live prices in USD, fetched in real time from the CoinGecko API
-Visualize your watchlist prices with a dynamic bar chart (Chart.js)
-Persistent storage using SQLite - your watchlist is saved between sessions
-Clean dark-themed UI

## Tech Stack  
- **Backend:** Python, Flask
- **Database:** SQLite
- **API:** CoinGecko API (free, no API key required)
- **Frontend:** HTML, CSS, Jinja2 templating
- **Visualization:** Chart.js
